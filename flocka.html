Flocka Time

Description

Flock Time is a Virtual Studio Code (VSC) extension and clone of the WakaTime productivity tool. The application allows the user to track the amount of hours he/she spends coding on a daily and weekly basis.

How it Works

The user can sign in via VSC, Facebook or by Email. When creating a profile, they are granted a unique API Key that they can then plugin to VSC so that we can track their productivity. Everytime the user hits save, the app tracks the time spent between each save and attributes that timestamp to the total number.

Technology Used

HTML, CSS, Bootstrap, Firebase, Javascript, JQuery, MomentJS
